# Streamlit CSV Test Writer

Einfache Streamlit-App, mit der du eine CSV-Datei hochladen kannst. Die App schreibt eine Spalte `Test` mit dem Wert `Test` in die Datei und bietet die bearbeitete CSV zum Download an.

## Setup

```bash
pip install streamlit pandas
```

## Start

```bash
streamlit run app.py
```

## Nutzung

1. CSV-Datei hochladen
2. Vorschau ansehen
3. Bearbeitete CSV herunterladen
